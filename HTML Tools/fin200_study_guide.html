FIN 200 Excel Memo

Chapter-by-Chapter Excel Practice Memo

This file is organized by chapter. Each chapter has its own narrow spreadsheet where the student types the inputs and writes the Excel formula directly. The goal is to practice the exact spreadsheet habit needed in finance questions.

How to use this memo
  • Each chapter is separated clearly.
  • Type the given numbers in the yellow input cells.
  • Write the Excel formula yourself in the blue formula row.
  • Use the green result row as the final output area.
  • This is meant to feel like a compact workbook, not a full Excel clone.

Chapter 2: Future Value of an Investment

Practice how to set up a simple future value problem in a compact spreadsheet layout.

FV Setup

Question Style

You invest $2,400 today at 7% annually for 6 years. Find the future value.

What to type

  • Inputs: present value, rate, years
  • Formula: Excel FV function
  • Result: final future value
Spreadsheet for Chapter 2
Cell Label Entry Meaning
B2 Present Value Input the amount invested today
B3 Rate Enter as decimal, e.g. 0.07
B4 Years Number of periods
B5 Excel Formula Example: =FV(B3,B4,0,-B2,0)
B6 Final Result Future value output

Chapter 3: Present Value of Multiple Cash Flows

Set up a row-by-row discounted cash flow spreadsheet.

Multiple CF

Question Style

Cash flows: 500 at year 1, 900 at year 2, and 1,400 at year 4. Discount rate = 10%.

What to type

  • One cash flow per row
  • One Excel formula per discounted row
  • One final SUM row
Spreadsheet for Chapter 3
Cell Label Entry Meaning

F4 Cash Flow at t=1 Year 1 cash flow
G4 Formula for PV at t=1 Example: =F4/(1+$F$2)^1

G5 Formula for PV at t=2 Example: =F5/(1+$F$2)^2
F6 Cash Flow at t=4 Year 4 cash flow
G6 Formula for PV at t=4 Example: =F6/(1+$F$2)^4
